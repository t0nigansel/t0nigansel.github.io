# tgansel.github.io

Personal GitHub Pages website for Toni Gansel.

The site is intentionally static: edit `index.html` and `styles.css`, then push to
`main`. GitHub Pages can serve this repository directly from the branch root.

## Local preview

Open `index.html` in a browser, or run a tiny local server:

```sh
python3 -m http.server 8080
```
